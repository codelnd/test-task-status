## Как запустить:
Клонировать репозиторий, транспилировать код и запустить в терминале. Результат выводится в логи.
```
git clone https://github.com/codelnd/test-task-status.git
cd test-task-status/src
tsc
node app.js
```

## CLI:
```
tsc // Транспиляция кода в JS
node app.js // Запуск
```
